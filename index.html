React useEffect Cleanup Pattern

JavaScript
by JohnDev
April 10, 2025
Code:
useEffect(() => {
  const subscription = someAPI.subscribe();
  
  return () => {
    subscription.unsubscribe();
  };
}, [dependency]);

Purpose/Goal

This pattern demonstrates how to properly clean up side effects in React's useEffect hook to prevent memory leaks.

Usage Example

For instance, when subscribing to external data sources or setting up event listeners that need to be removed when the component unmounts.

Key Concepts/Why it Works

The useEffect hook accepts a function that can return a cleanup function. React will call this cleanup function before the component unmounts or before re-running the effect due to a dependency change.

Potential Gotchas/Alternatives

Forgetting to clean up subscriptions can cause memory leaks. Make sure dependency arrays are correctly specified to avoid stale closures.
